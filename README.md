# 📊 Expense Dashboard | Microsoft Excel + VBA

## 📌 Project Overview

This project is a fully interactive **Expense Dashboard** built in **Microsoft Excel** using **VBA Automation, Pivot Tables, Pivot Charts, Slicers, and Excel Tables**.

The dashboard transforms raw expense data into meaningful business insights. It helps analyze spending patterns, compare budgets with actual expenses, monitor approval rates, and support data-driven business decisions.

This project was created to strengthen my Business Analyst and Data Analyst skills through hands-on dashboard development.

---

## 🚀 Dashboard Features

- Fully Automated Dashboard using VBA
- Interactive Month and Department Slicers
- Professional KPI Cards
- Expense by Department Analysis
- Expense by Category Analysis
- Monthly Expense Trend
- Budget vs Actual Comparison
- Payment Method Analysis
- Automatic Pivot Table Generation
- Automatic Chart Creation
- Dynamic Excel Table
- One-Click Dashboard Refresh
- Clean and Professional Dashboard Design

---

## 📈 Key Performance Indicators (KPIs)

- Total Expenses
- Budget Allocated
- Budget Remaining
- Total Transactions
- Approval Rate
- Average Expense per Transaction

---

## 📊 Business Questions Answered

This dashboard helps answer important business questions such as:

- Which department spends the most?
- Which expense category has the highest spending?
- Is the company staying within budget?
- How much budget is still available?
- Which payment method is used the most?
- What is the monthly spending trend?
- What percentage of expenses are approved?
- What is the average expense per transaction?
